Описание программы
Weather Diary - это графическое приложение для ведения дневника наблюдений за погодой. Программа позволяет:
- Добавлять записи о погоде с указанием даты, температуры, описания и наличия осадков
- Просматривать все записи в удобной табличной форме
- Фильтровать записи по дате и температуре
- Сохранять данные в JSON файл и загружать из него
- Удалять ненужные записи

Системные требования
- Python 3.7 или выше
- Операционная система: Windows / macOS / Linux
